# Contributing to ShuttleChamp

We welcome contributions from the community. Please read the following guidelines to get started.

## How to Contribute

1. **Fork the Repository**
   - Navigate to the ShuttleChamp GitHub page and fork the repository.

2. **Clone Your Fork**
   ```bash
   git clone https://github.com/yourusername/ShuttleChamp.git
   cd ShuttleChamp
   ```

3. **Create a Branch**
   - Use a descriptive name for your branch.
   ```bash
   git checkout -b feature/your-feature-name
   ```

4. **Make Changes**
   - Implement your changes and ensure your code follows the project's coding standards.

5. **Commit Changes**
   - Use clear and descriptive commit messages.
   ```bash
   git commit -m 'Add some feature'
   ```

6. **Push Changes**
   ```bash
   git push origin feature/your-feature-name
   ```

7. **Create a Pull Request**
   - Navigate to your fork on GitHub and create a pull request against the main repository.

## Code Style

- Follow PEP 8 for Python code
- Use Prettier for formatting JavaScript and TypeScript code

## Testing

- Ensure all tests pass before submitting a pull request
- Add tests for new features and bug fixes
